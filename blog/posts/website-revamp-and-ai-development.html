Website Revamp and AI Development Experience

When I created this website, I did so because I didn't have much professional experience and I thought it would help landing my first jobs. Unfortunately, it's been sitting there without any changes and it was due a revamp. I only wanted to make it a bit more personal and also make it more useful.

I wanted to start the habit of writing more often and I thought adding a blog would be a nice addition. Regarding the technology stack, I didn't want to complicate things at all - and I truly mean that. This site is just plain HTML and CSS with some JavaScript.

I reckon there are some things that come for free when you use something like a static site generator. But in my case, I'm just using Cursor, which is an AI-assisted IDE. So for example, for adding this blog post, I'm actually just narrating all of this and Cursor will create the HTML page and update the rest of the website to link to it. In that way, it reduces friction to create content.

I do acknowledge that this creates a huge vendor lock-in with this specific tool. But honestly, just interacting with it for many different use cases has helped me understand a lot about the possibilities of foundational models and what can be done with them.

Even if I'm just using a simple interface as it is, Cursor with its agent capability makes me see potential applications of this technology. So I will keep using this in this way so that I keep learning about it.
